In this project the database is constructed in XAMPP.

so first install XAMPP then create a database.
on that following database create a table named "students" (without double cotetion)
after that run this SQL command to create the "students" table.

CREATE TABLE students(
  id varchar(7),
  name varchar(20),
  phone varchar(7),
  password varchar(40),
  phone varchar(11),
  cgpa double(3,2)
);

now the database is ready!!!
start the XAMPP application and run the LoginPage.java program.
